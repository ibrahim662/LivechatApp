## How to run it in local?
+ Clone this project ( $~git@github.com:ibrahim662/Weather-Website.git ) 

### Open two teminals ( for Client and Server side )
#### Client side

+ $~cd client/
+ $~npm install
+ $~npm start

### Second terminal 
#### Server side
+ $~cd server/
+ $~npm install
+ $~npm start
